A Hands-On RDKit Tutorial آموزش عملی RDKit

RDKit is the leading open-source toolkit for cheminformatics. This guide takes you from installation to real molecular analysis in Python. RDKit پرکاربردترین کتابخانه‌ی متن‌باز برای شیمی‌انفورماتیک است. این راهنما شما را از نصب تا تحلیل واقعی مولکولی در پایتون همراهی می‌کند.

Python RDKit Cheminformatics

1Introductionمقدمه

RDKit lets you represent, manipulate, and analyze chemical molecules programmatically. It's widely used in drug discovery, molecular property prediction, and machine learning for chemistry. In this tutorial you'll learn how to build molecules from SMILES strings, compute chemical properties, search for substructures, measure molecular similarity, and render 2D structures.

RDKit به شما اجازه می‌دهد مولکول‌های شیمیایی را به‌صورت برنامه‌نویسی نمایش داده، دستکاری و تحلیل کنید. این کتابخانه به‌طور گسترده در کشف دارو، پیش‌بینی ویژگی‌های مولکولی و یادگیری ماشین برای شیمی استفاده می‌شود. در این آموزش یاد می‌گیرید چگونه از رشته‌های SMILES مولکول بسازید، ویژگی‌های شیمیایی را محاسبه کنید، زیرساختارها را جستجو کنید، شباهت مولکولی را اندازه بگیرید و ساختارهای دوبعدی را رسم کنید.

💡 Prerequisite: Basic Python knowledge. No prior chemistry background is strictly required, though it helps. 💡 پیش‌نیاز: آشنایی مقدماتی با پایتون. داشتن پیش‌زمینه‌ی شیمی الزامی نیست، اما کمک‌کننده است.

2Installationنصب

The easiest way to install RDKit is with pip or conda. Conda is historically the most reliable, but recent pip wheels work well too.

ساده‌ترین راه نصب RDKit استفاده از pip یا conda است. conda از نظر تاریخی قابل‌اعتمادترین روش بوده، اما نسخه‌های اخیر pip نیز به‌خوبی کار می‌کنند.

bash 
# Using pip
pip install rdkit

# Or using conda (recommended for full features)
conda install -c conda-forge rdkit

3Creating Moleculesساخت مولکول

Molecules are usually created from SMILES (Simplified Molecular Input Line Entry System) strings — a compact text notation for chemical structures.

مولکول‌ها معمولاً از رشته‌های SMILES ساخته می‌شوند — یک نشانه‌گذاری متنی فشرده برای ساختارهای شیمیایی.

python 
from rdkit import Chem

# Create a molecule from a SMILES string (this is caffeine)
smiles = "CN1C=NC2=C1C(=O)N(C(=O)N2C)C"
mol = Chem.MolFromSmiles(smiles)

# Check it parsed correctly
print(mol is not None)          # True
print(mol.GetNumAtoms())        # number of (heavy) atoms
print(Chem.MolToSmiles(mol))    # canonical SMILES
⚠️ If a SMILES string is invalid, MolFromSmiles returns None instead of raising an error — always check for it. ⚠️ اگر رشته‌ی SMILES نامعتبر باشد، MolFromSmiles به‌جای خطا مقدار None برمی‌گرداند — همیشه آن را بررسی کنید.

4Basic Propertiesویژگی‌های پایه

Once you have a molecule object, you can iterate over its atoms and bonds and inspect their properties.

پس از ساخت یک شیء مولکول، می‌توانید روی اتم‌ها و پیوندهای آن پیمایش کرده و ویژگی‌هایشان را بررسی کنید.

python 
from rdkit import Chem

mol = Chem.MolFromSmiles("CCO")   # ethanol

# Loop over atoms
for atom in mol.GetAtoms():
    print(atom.GetIdx(), atom.GetSymbol(), atom.GetDegree())

# Loop over bonds
for bond in mol.GetBonds():
    print(bond.GetBeginAtomIdx(), "-", bond.GetEndAtomIdx(),
          bond.GetBondTypeAsDouble())

5Molecular Descriptorsتوصیفگرهای مولکولی

Descriptors are numerical values that summarize a molecule — molecular weight, LogP, hydrogen-bond donors/acceptors, and more. They're the foundation of QSAR and ML models.

توصیفگرها مقادیر عددی هستند که یک مولکول را خلاصه می‌کنند — وزن مولکولی، LogP، دهنده/گیرنده‌های پیوند هیدروژنی و موارد دیگر. آن‌ها پایه‌ی مدل‌های QSAR و یادگیری ماشین‌اند.

python 
from rdkit import Chem
from rdkit.Chem import Descriptors

mol = Chem.MolFromSmiles("CC(=O)OC1=CC=CC=C1C(=O)O")  # aspirin

print("MolWt:", Descriptors.MolWt(mol))
print("LogP:", Descriptors.MolLogP(mol))
print("H-bond donors:", Descriptors.NumHDonors(mol))
print("H-bond acceptors:", Descriptors.NumHAcceptors(mol))
print("TPSA:", Descriptors.TPSA(mol))

6Substructure Searchجستجوی زیرساختار

You can check whether a molecule contains a particular chemical pattern using SMARTS queries — extremely useful for filtering compound libraries.

می‌توانید بررسی کنید که آیا یک مولکول شامل الگوی شیمیایی خاصی هست یا نه، با استفاده از پرس‌وجوهای SMARTS — بسیار کاربردی برای فیلتر کردن کتابخانه‌های ترکیبات.

python 
from rdkit import Chem

mol = Chem.MolFromSmiles("CC(=O)OC1=CC=CC=C1C(=O)O")  # aspirin

# Define a carboxylic acid pattern as SMARTS
pattern = Chem.MolFromSmarts("C(=O)[OH]")

# Does the molecule contain it?
print(mol.HasSubstructMatch(pattern))    # True

# Get the matching atom indices
print(mol.GetSubstructMatches(pattern))

7Fingerprints & Similarityاثرانگشت و شباهت

Fingerprints encode a molecule as a bit vector, letting you compute how similar two molecules are — typically with the Tanimoto coefficient.

اثرانگشت‌ها یک مولکول را به‌صورت بردار بیتی رمزگذاری می‌کنند و به شما امکان می‌دهند شباهت دو مولکول را محاسبه کنید — معمولاً با ضریب تنیموتو.

python 
from rdkit import Chem
from rdkit.Chem import AllChem, DataStructs

mol1 = Chem.MolFromSmiles("CCO")       # ethanol
mol2 = Chem.MolFromSmiles("CCN")       # ethylamine

# Morgan (circular) fingerprints, radius 2
fp1 = AllChem.GetMorganFingerprintAsBitVect(mol1, radius=2, nBits=2048)
fp2 = AllChem.GetMorganFingerprintAsBitVect(mol2, radius=2, nBits=2048)

# Tanimoto similarity (0 = different, 1 = identical)
similarity = DataStructs.TanimotoSimilarity(fp1, fp2)
print("Tanimoto similarity:", similarity)

8Drawing Moleculesرسم مولکول

RDKit can render 2D depictions of molecules — perfect for notebooks, reports, and this very tutorial.

RDKit می‌تواند نمایش‌های دوبعدی از مولکول‌ها تولید کند — عالی برای نوت‌بوک‌ها، گزارش‌ها و همین آموزش.

python 
from rdkit import Chem
from rdkit.Chem import Draw

mol = Chem.MolFromSmiles("CC(=O)OC1=CC=CC=C1C(=O)O")  # aspirin

# Save a single molecule image
Draw.MolToFile(mol, "aspirin.png", size=(400, 300))

# Draw a grid of several molecules
mols = [Chem.MolFromSmiles(s) for s in ["CCO", "CCN", "c1ccccc1"]]
img = Draw.MolsToGridImage(mols, molsPerRow=3,
                           legends=["ethanol", "ethylamine", "benzene"])
img.save("grid.png")
